# Security Policy

This repository may discuss malware indicators and firmware security issues.

## Reporting Issues

If you find an error, missing context, incorrect claim, or additional evidence related to this firmware analysis, please open a GitHub issue or pull request with reproducible details.

Useful information includes:

- firmware filename and hash
- affected file path inside the firmware
- command used to reproduce the finding
- relevant hash, string, offset, or extracted artifact metadata
- whether the evidence comes from static or dynamic analysis

## Handling Potential Malware

Do not submit live malware samples without clear labeling. Do not post instructions intended to help deploy, persist, evade detection, or operate malware.

Binary samples should be treated as potentially malicious and should be shared only when necessary for defensive research, with hashes and safety warnings.

## Vendor Reporting

If this analysis is used for vendor/service-center reporting, include only factual and reproducible claims. Avoid attributing intent unless there is direct evidence.
