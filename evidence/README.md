# Evidence Directory

This directory stores small, text-based evidence files for reproducibility.

Do not place large firmware images or live malware binaries here unless they are clearly required, safely labeled, and allowed by the hosting platform.

Recommended evidence files:

```text
hashes-known.txt
unique-indicator-search-results.txt
binwalk output
strings excerpts
payload hashes
partition list
mount list
command logs
```

Binary firmware samples should be handled separately with hashes and safety warnings.
